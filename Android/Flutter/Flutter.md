`pubspec.yaml`：包模块配置文件

```

	pub:
```

#### pubspec.yaml
```

	name:
	environment:
	dependencies:
	flutter:
		assets:
		uses-material-design:
```


### flutter
```

	devices:
	run:
		-d: 指定设备
```
